## Getting Started
1. Clone the repo:
   ```bash
   git clone https://github.com/Akshith-cdr/RYZEx-AI.git
   cd RYZEx-AI
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment
Build for production:
```bash
npm run build
```
